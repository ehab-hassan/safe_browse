# Safe Browse — React

React version of the Safe Browse encrypted terminal UI.

## Setup

```bash
cd react-app
npm install
```

Copy your logo into `public/images/logo.png` (create the `images` folder if needed).

## Run

```bash
npm run dev
```

Open http://localhost:5173 in your browser.

## Build

```bash
npm run build
```

Output is in `dist/`. Serve with `npm run preview` to test the production build.
